弹出消息控件。

包括提示、警告、错误等三种类型。

引入ui.msg模块，然后就可以使用Msg对象来创建消息了。

```html
<script>
  require(['ui.msg'], function(Msg){
    Msg('这是一条提示信息');
  });
</script>
```

###提示信息

```js
Msg('这是一条提示信息');
```

[查看效果](#showMsg)

###警告信息

```js
Msg.warn('这是一条警告信息');
```

[查看效果](#showWarn)

###错误信息

```js
Msg.error('这是一条错误信息');
```

[查看效果](#showError)
